# substring-search

Проект по поиску подстроки в строке: реализация точных и нечётких алгоритмов.

## Структура

- `subsearch/` — библиотека алгоритмов поиска подстрок (точные и нечёткие).
- `benchmark/` — приложение для тестирования производительности.

## Сборка

```bash
mkdir build && cd build
cmake ..
cmake --build .
